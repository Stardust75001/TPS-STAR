# TPS STAR Workflows Cheatsheet

| Workflow Name      | Function (Detailed Description)                                                                 |
|--------------------|-----------------------------------------------------------------------------------------------|
| send-email.yml     | Sends notification emails based on workflow triggers. Validates i18n JSON and compares keys.   |
| build-data-extraction.yml | Automates extraction and transformation of analytics data.                              |
| data-quality-check.yml | Runs data validation checks before analytics processing.                                   |
| weekly-analytics-report.yml | Schedules and generates weekly analytics reports.                                     |
| notify-on-failure.yml | Sends notifications when a workflow fails.                                                  |
| ...                | Add new workflows here as they are created.                                                    |
