# TPS STAR Scripts Cheatsheet

| Script Name        | Function (Detailed Description)                                                                 |
|--------------------|-----------------------------------------------------------------------------------------------|
| sync-locales.js    | Synchronizes locale JSON files by adding missing keys from en.default.json to all other locales. |
| build-data-extraction.sh | Extracts and transforms raw data for analytics processing.                                 |
| data-quality-check.py | Validates data integrity and completeness before analysis.                                   |
| weekly-analytics-report.py | Generates weekly analytics reports for business KPIs.                                   |
| send-email.py      | Sends automated email notifications based on report results.                                   |
| ...                | Add new scripts here as they are created.                                                      |
